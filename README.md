# GTT Center Perth

**WA's first premium waiting experience for pregnant women undergoing the Glucose Tolerance Test.**

Pregnant women spend up to 3 hours at pathology for the GTT (Gestational Diabetes Test). GTT Center Perth transforms that wait into a restorative experience: massage, nails, hair, and 3D keepsake ultrasound — all under one roof.

## Model

- **Pathology:** Partner with PathWest/WDP for on-site blood collection (no NATA accreditation required)
- **Services:** Sublet chairs/rooms to self-employed practitioners (massage, nails, hair)
- **3D Scan:** Sublet scan room to existing Perth keepsake operator
- **Entity:** YETI Holding Trust
